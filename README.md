# Tarefas — Alice & Carolina

PWA de tarefas domésticas para Alice e Carolina.

## Como publicar no GitHub Pages

1. Vai a https://github.com/new
2. Cria um repositório com o nome: **tarefas**
   - Visibilidade: Public
   - Não adicionar README (o projeto já tem ficheiros)
3. Faz upload de todos os ficheiros desta pasta para o repositório
4. Vai a Settings → Pages → Source: seleciona "main" e pasta "/ (root)"
5. A app fica disponível em: https://ftoipa.github.io/tarefas

## Instalar no telemóvel (Android)

1. Abre o Chrome e vai ao link acima
2. Aparece um banner "Adicionar ao ecrã inicial" — aceita
3. A app instala-se como qualquer outra

## Instalar no telemóvel (iPhone)

1. Abre o Safari e vai ao link acima
2. Toca no botão de partilha (caixa com seta para cima)
3. Seleciona "Adicionar ao ecrã de início"
